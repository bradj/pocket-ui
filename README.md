# Pocket UI

Web interface for [Pocket](https://github.com/bradj/pocket)

## Getting Started

1. Follow **Pocket's** [Getting Started](https://github.com/bradj/pocket#getting-started). **Pocket UI** requires an API.
1. Clone this repo
1. `npm install`
1. `npm run build`
1. `npm start` or `npm run dev`
